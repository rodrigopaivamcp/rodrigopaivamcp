Exemplo 8 HTML - Uso das tags <ol>,<,ul> e <li>

As tags em <ol>, <ulo> representão listas em html, e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada ja a tag <ul> representa uma lista não ordenada.

Em ambas as listas para representar os elementos da lista usamos a tag <li>

Além disso, esas listas podem conter outras listas de forma alinhadas.

Etapas para desenvolver um site(Lista Ordenada)

  1. Planejamento
    1. Definir o objeyivo do site
    2. Identificar os públic alvo
    3. Criar um mapra do site(site map)
  2. Desing
    1. Criar wireframes e protótipos
    2. Definir a paleta de cores e tipografia
    3. Desenvolver o layout no Figma ou Adode XD
  3. Desenvolvimento
    1. Configurar o ambiente de desenvolvimento.
    2. Codificar o front-end(thm, css, e javascript)
    3. Implementar o back-end, com linguagens orientadas a objeto(C#, Java, PHP,Python)
  4. Testes
    1. Testar a responsividade em diferente dispositivos
    2. Realizar testes de usabilidade
    3. Corrigir bugs identificados
  5. Publicação
    1. Escolher um provedor de hospedagem
    2. Consifgurar o domínio e o SSL
    3. Fazer o deploy do site.
