Estrutura de um documento HYML

A estrutura de u documento HTML é definida pela declaração !DOCTYPE html , pelas tags html, heaqd e body.

Essa estrutura é interpretada pelo navegador que analisa cada tag e renderiza para o usuário.

A tag heasd é utilizada para configurações do navegador e consequentemente do documento HTML, as informações contidas dentro dessa tag são invisíveis para o usuário, com exceçpão do título que ficará visiível na parte superior da página.

Ja a tag body é utilizada para exibir toda parte viível para o usuário e nela teremos outras tags que também serão utilizadas pelo navegador.

Tag HTML, essa tag pe a raiz que contém todas as outras tags do documento html, delimitando a totalidade do documento.

Por fim a instrução DOCTYPE html, que tem a fuinalidade de indicar ao navegador qual a versão do html que será utilizado. Ela deve vir na primeira linha do documento.
